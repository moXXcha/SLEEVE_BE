# 機能一覧書（ログイン要件別整理版）

本ドキュメントは、本サービスにおける機能を **ログイン不要機能 / ログイン必須機能 / 管理者機能** に分類して整理したものです。  
各機能には、関連する要件（AI要件 / インフラ要件 / 実装済み）を併記します。  

---

## 1. ログイン不要で利用可能な機能

未ログインのユーザーが利用できる機能。主に **レコメンド表示・検索・閲覧** など、導入体験を目的とする機能が中心となる。

### 1.1 推薦・検索系
- コーデおすすめ取得（AI要件あり）  
- itemおすすめ取得（AI要件あり）  
- tagおすすめ取得（AI要件あり）  
- searchおすすめ取得（AI要件あり）  
- 検索機能（AI要件あり）  

### 1.2 閲覧系
- コーデ詳細取得（実装済み / インフラ要件）  
- item詳細取得（インフラ要件）  
- 相場取得  
- 該当アイテム使用のコーデ取得（インフラ要件）  
- profile閲覧（インフラ要件）  

### 1.3 認証・登録
- email user登録  
- google アカウント連携  
- apple アカウント連携  
- line アカウント連携  
- x アカウント連携  

---

## 2. ログイン必須な機能

ログイン後にのみ利用できる機能群。  
主に **投稿・購入・編集・通報** などの「ユーザーアクション系」が含まれる。  

### 2.1 アカウント管理
- ログアウト  
- mfa認証  
- user follow  
- profile作成（インフラ要件）  
- profile編集（インフラ要件）  
- user summary取得（実装済み / インフラ要件）  

### 2.2 コーデ関連
- コーデシェア  
- コーデ投稿（インフラ要件）  
- コーデ投稿編集（インフラ要件）  
- コーデ削除  
- コーデ削除一覧取得（インフラ要件）  
- コーデ削除取り消し  
- コーデ下書き（インフラ要件）  

### 2.3 アイテム関連
- item購入  
- item出品（インフラ要件）  
- item出品編集（インフラ要件）  
- item出品削除  
- 出品取り消し  
- item下書き（インフラ要件）  

### 2.4 通知・お知らせ
- お知らせ一覧取得  
- お知らせ詳細取得  
- 通報  
- user alert機能  

### 2.5 決済関連
- 口座登録  
- 口座削除  
- credit登録  
- credit削除  

---

## 3. 管理画面機能（管理者専用）

管理者（CS / 運用担当）が利用する機能。  
主に **モデレーション・ユーザー管理・お知らせ配信** を担う。  

- 通報一覧  
- 通報詳細  
- 通報検索  
- userBAN  
- お知らせ送信  
- お知らせ削除  
- user検索  
- item検索  
- code検索  
- 各項目削除機能  

---

## 4. 実装済み機能

すでに実装済みで稼働している機能。  
現時点でのサービス基盤を支えている。  

- コーデ詳細取得（インフラ要件）  
- user summary取得（インフラ要件）
