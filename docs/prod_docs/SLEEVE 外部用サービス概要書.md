## **1. サービス概要**
- **サービス名**  
  **SLEEVE**

- **スローガン**  
	「あの人らしいが自分らしいになるところ。」  

- **サービス説明**  
  コーデ投稿とアイテム売買を融合させたCtoC型ファッションリユースプラットフォーム。  
  誰でも簡単に「マネキン買い」ができ、リユース市場で手軽におしゃれを楽しめる体験を提供します。

- **ビジョン・コンセプト**  
「若者がファッションに触れる、リユースがファッションの主要市場になる袖口に」
「手軽に手頃に、日常を少しおしゃれに」
  → ファッションをもっとカジュアルで身近にし、個人が自己表現できる場所を創出。

## **3. 提供価値**
- **ユーザー体験**
  - コーデ投稿で「自己表現」＆「収益化」チャンス
  - 気に入ったコーデを**そのままマネキン買い**
  - リユース品や類似品で**低コストでトレンド再現**

- **解決する課題**
  - 「何を着たらいいか分からない」「新品は高い」  
  → コーデ提案＋CtoCの価格優位性で解決

- **差別化要素**
  - メルカリにはない**「コーデベースの購入体験」**
  - ZOZOにはない**「CtoCによる低価格」**
